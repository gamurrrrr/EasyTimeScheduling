### Step 0: Clone and pull latest changes
clone: 
1. git clone https://github.ubc.ca/COSC-O-305-101-O2025W1/L03_Google.git
2. cd L03_Google

Pull from dev (NOT main, main is for final clean project): 
3. git checkout dev
4. git pull origin dev


### Step 1: Install Node.js
1. go to: https://nodejs.org/zh-cn/download, for Windows, download the LTS versiom msi file.
2. run installer, keep default
3. check version:
node -v
npm -v


### Step 2: go to frontend folder
cd client


### Step 3: Install dependencies
npm install


### Running the dev server
npm run dev
